# Notification Sound Placeholder

Este directorio debe contener un archivo de sonido para las notificaciones.

## Archivo requerido:
- `notification.mp3`

## Características recomendadas:
- Duración: 1-3 segundos
- Formato: MP3 o WAV
- Volumen: Moderado
- Tono: Agradable y no estridente

## Fuentes sugeridas para sonidos:
- [Freesound.org](https://freesound.org)
- [ZapSplat](https://www.zapsplat.com)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## Instrucciones:
1. Descarga un sonido de notificación
2. Nómbralo como `notification.mp3`
3. Colócalo en el directorio `assets/sounds/`
4. El sonido se reproducirá automáticamente al finalizar cada ciclo

## Nota:
Si no se proporciona un archivo de sonido personalizado, la app usará el sonido predeterminado del sistema para las notificaciones.
