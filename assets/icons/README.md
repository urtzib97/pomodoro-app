# Icons Directory

Este directorio está reservado para iconos personalizados de la aplicación.

## Uso:
Los iconos se pueden usar para:
- Estados del temporizador
- Ilustraciones en pantallas vacías
- Iconos de notificaciones personalizados

## Iconos del sistema:
La app utiliza Material Icons por defecto, que incluye:
- `timer` - Temporizador
- `task_alt` - Tareas
- `bar_chart` - Estadísticas
- `settings` - Configuración
- `check_circle` - Completado
- `play_arrow` - Iniciar
- `pause` - Pausar
- `refresh` - Reiniciar

No es necesario agregar iconos personalizados a menos que quieras un diseño más personalizado.
